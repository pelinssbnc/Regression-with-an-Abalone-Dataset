# 🐚 Abalone Yaş Tahmini (Regression with Abalone Dataset)

Deniz salyangozunun fiziksel ölçümlerini kullanarak
yaşını tahmin eden makine öğrenmesi regresyon projesi.

---

## 📌 Genel Bakış

Abalone'nin yaşını belirlemek zahmetli bir işlemdir.
Bu projede kabuk ölçümleri gibi kolay elde edilebilen
özelliklerden yaş tahmini yapılmaktadır.

Temel hedefler:
- Özellikler ile yaş arasındaki korelasyonu analiz etmek
- Regresyon modellerini eğitmek ve karşılaştırmak
- MAE ve RMSE metrikleriyle sonuçları değerlendirmek
